## 📖 О проекте

**"Грань Забвения"** — это игра-кликер в жанре выживания, где вам предстоит управлять отрядом выживших в разрушенном мире. Собирайте ресурсы, улучшайте базу, сражайтесь с противниками и выполняйте квесты, чтобы выжить в жестоком постапокалипсисе.

Проект создан на **Godot Engine 4** и распространяется под лицензией **MIT**.

## ✨ Особенности

### 🎮 Основной геймплей
- **Сбор ресурсов** — добывайте еду, воду и материалы в разных локациях
- **Система улучшений** — покупайте апгрейды для увеличения эффективности сбора
- **Смена дня и ночи** — со временем ресурсы расходуются, а здоровье падает

### 👥 Система выживших
- **Найм и управление** — собирайте отряд из разных специализаций
- **Назначение работ** — распределяйте выживших на сбор ресурсов или охрану
- **Боевая система** — только охранники участвуют в сражениях
- **Постоянная смерть** — погибшие выжившие исчезают навсегда

### ⚔️ Боевая система
- **Тактические сражения** — управляйте отрядом в автоматических боях
- **Разные типы врагов** — крысы, бандиты, зомби, мутанты и генералы
- **Система отрядов** — противники собираются в группы с лидерами
- **Регулировка скорости** — ускоряйте или замедляйте бой

### 🗺️ Карта мира
- **Разные регионы** — лес, горы, водоемы с уникальными ресурсами
- **NPC и диалоги** — общайтесь с персонажами, получайте квесты
- **Квестовая система** — выполняйте задания для получения наград

### 🏆 Достижения и квесты
- **Разнообразные достижения** — собирайте ресурсы, убивайте врагов, выживайте
- **Цепочки квестов** — открывайте новые задания по мере прогресса
- **Многоэтапные задания** — сложные квесты с несколькими стадиями

### 🛡️ Режим Бога
- **Чит-панель** — вызывается комбинацией `Shift+Q`
- **Добавление ресурсов** — ускорение прогресса для тестирования
- **Призыв выживших и врагов** — мгновенное создание любых юнитов
- **Режим неуязвимости** — бессмертие для исследования механик


## 🚀 Установка

### Системные требования

- **Операционная система**: Windows 10/11, Linux (любой дистрибутив)

- **Оперативная память**: 2 GB
- **Видеокарта**: Intel HD Graphics 4000 или аналог

📝 Создание контента
Добавление нового улучшения

    Создайте файл .tres в папке Resources/upgrades/

    Назначьте скрипт upgrade_resource.gd

    Заполните параметры:

        upgrade_id — уникальный ID

        title — название

        cost — цена в материалах

        upgrade_type — тип эффекта

Добавление нового врага

    Создайте файл .tres в папке Resources/enemies/

    Назначьте скрипт enemy_data.gd

    Настройте характеристики

    При необходимости создайте группу в enemies_groups/
